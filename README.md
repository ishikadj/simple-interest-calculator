# Simple Interest Calculator

## Description
This project takes three inputs: the principal amount, the annual interest rate, and the time period in years. It then outputs the total simple interest earned.

## Calculation Formula
The calculator uses the standard simple interest formula:
$$A = P(1 + rt)$$

Where:
* **P** = Principal amount
* **r** = Annual interest rate (decimal)
* **t** = Time (years)

## How to Use
1. Enter the **Principal** amount.
2. Enter the **Interest Rate** per year.
3. Enter the **Time** period in years.
4. Click 'Calculate' to see the result.
